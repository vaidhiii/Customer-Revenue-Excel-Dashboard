# Customer-Revenue-Excel-Dashboard

## Project Overview  
This project is an interactive sales analytics dashboard built in Microsoft Excel to analyze customer behavior and revenue performance for January 2024.

## What the Dashboard Shows  
The dashboard includes:

- Total Revenue by City  
- Total Revenue by Product  
- Revenue Share by Customer Type  
- Daily Revenue Trend (Jan 2024)  
- Interactive slicers for City and Customer Type  

Users can filter the visuals dynamically using slicers.

## Tools Used  
- Microsoft Excel  
- Pivot Tables  
- Pivot Charts  
- Slicers  
- Data Cleaning & Formatting  

## Dataset  
The dataset contains:
- Customer ID  
- Date  
- City  
- Customer Type (New, Returning, VIP)  
- Product  
- Category  
- Quantity  
- Unit Price  
- Total Sales  

## Objective  
To build an interactive, clean, and professional Excel dashboard that helps understand:
- Which city generates the most revenue  
- Which product sells best  
- How different customer types contribute to sales  
- How sales change daily
